# Disclaimer


1. Coding agents don't actually have senses and are not sentient, with or without this tool. While the final output should align more to established patterns, coder discretion is still advised and you should absolutely always read the code.

2. Nothing in the branding/naming of the tool was intentionally copied. After a bit more investigation, some correlating concepts were found, which may have unintetionally and subconciously inspired the naming:
   1. Yann LeCunn's talks on World Models and how LLMs don't learn from the world how humans do. Highly recommend you check out one of his [talks on the subject](https://www.youtube.com/watch?v=2j78HCv6P5o&).
   2. [Refactoring Guru's "nose"](https://refactoring.guru/refactoring/smells) joke. The jokes in this repo were written before that one was discovered. 

3. No LLMs were used for the humor, but some may have been used to write the code, iterate on ideas, refactor and type.

4. The tool was made by a solo developer on a budget coding plan. If you plan to sue, please don't - there's no money for lawyers.
